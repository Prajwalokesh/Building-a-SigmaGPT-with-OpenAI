# Building a SigmaGPT with OpenAI

This project demonstrates how to build a SigmaGPT-style AI assistant using OpenAI APIs with a modular backend architecture.

## Features
- OpenAI-powered conversational backend
- Clean project structure for scalability
- Environment-based API key handling

## Tech Stack
- Python
- OpenAI API
- FastAPI / Flask (if used)
- JavaScript / React (frontend, optional)
